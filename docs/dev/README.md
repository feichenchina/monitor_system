# AI 服务器监控系统 - 开发文档索引

本目录记录了项目开发过程中的核心技术实现、决策逻辑及踩坑经验。通过这些文档，开发者可以快速了解系统的演进过程和底层原理。

## 核心文档概览

### 1. [项目初始化与重构日志](01-project-init-and-refactor.md)
- **核心内容**: 介绍项目从单体混乱代码向模块化架构的转变，以及标准化日志系统的建立。
- **技术要点**: FastAPI 结构、Python 异步日志轮转、目录规范化。

### 2. [Windows 服务化与启动脚本增强](02-windows-service-and-startup.md)
- **核心内容**: 解决 Windows 环境下的生产部署问题，实现静默运行、自启动及进程独占检测。
- **技术要点**: C# Service Wrapper 动态编译、PowerShell 权限管理、VBS 静默启动、端口占用检测。

### 3. [前端 UI/UX 优化与性能提升](03-ui-optimization.md)
- **核心内容**: 提升 Web 界面的流畅度和视觉质量，解决数据刷新时的闪烁问题。
- **技术要点**: Vue 3 "Smart Merge" 差量更新算法、CSS 布局优化、Favicon Base64 内联方案。

### 4. [IBMC 配置与机器管理功能增强](04-ibmc-and-machine-management.md)
- **核心内容**: 完善服务器带外管理（IBMC）集成，支持机器状态的精细化标记与数据批量操作。
- **技术要点**: SQLModel 数据库迁移、实时失焦保存逻辑、CSV 导入导出实现。

### 5. [昇腾 NPU 优化与异构机器监控](05-npu-and-heterogeneous-monitor.md)
- **核心内容**: 针对异构计算环境（NV GPU + Huawei NPU）的深度监控适配。
- **技术要点**: NPU 进程级状态判断逻辑、SSH 多命令合并执行优化、异构数据聚合模型。

### 6. [交互体验修复日志：剪贴板、焦点与密码显隐](06-interaction-and-clipboard-fixes.md)
- **核心内容**: 处理 Web 端复杂的交互冲突，特别是安全上下文受限环境下的功能补全。
- **技术要点**: 剪贴板 Fallback 降级、Vue 状态拦截、防止 `blur` 事件冲突的策略模式。

### 7. [项目版本管理与自动化工具](07-version-and-project-management.md)
- **核心内容**: 建立标准化的版本发布流程和同步工具。
- **技术要点**: 前后端版本注入、Python 自动化同步脚本。

### 8. [PCIe 拓扑可视化集成](08-pcie-topo-integration.md)
- **核心内容**: 实现复杂的硬件拓扑自动采集与图形化展示，并随底层子模块升级支持多链路与存储节点识别。
- **技术要点**: Cytoscape.js 渲染、远程自动化工具分发、沉浸式 UI 设计、Cytoscape 多重连线冲突处理方案。
